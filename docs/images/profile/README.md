# Profile Images

Place your profile image here.

## File Name
Name your profile image: `profile.jpg` (or `profile.png`, `profile.webp`)

## Supported Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Recommended Size
- Minimum: 400x400 pixels
- Recommended: 800x800 pixels or larger
- Aspect Ratio: 1:1 (square) for best results

## Usage
The image will automatically be used on the home page. If the image fails to load, a placeholder will be shown instead.
